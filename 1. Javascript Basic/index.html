Javascript Basics

Javascript is a lightweight, cross-platform, object-oriented computer programming language.


Lightweight

It means the language doesn't eat up much memory of computer. It has relatively simple syntax and features.

Cross Platform

It means the language can be used in multiple platform.

Object Oriented

It means the language that is based on object.
Today Javascript can be used in different places
  • Client Side: Javascript was traditionally only used in the browser
  • Server Side: We can use Javascript on the server as well by Node.js
